# 🎯 Object Detection and Tracking using YOLOv8

## 📌 Project Overview

This project performs real-time object detection and tracking using **YOLOv8** and **OpenCV**.

The application captures live video from a webcam or video file, detects objects, assigns tracking IDs, draws bounding boxes, and displays the results in real time.

---

## 🚀 Features

- 🎥 Real-time webcam detection
- 📹 Video file detection
- 🎯 Object Tracking
- 🏷️ Object Labels
- 🆔 Tracking IDs
- 📦 Bounding Boxes
- 💾 Output video saving

---

## 🛠️ Technologies Used

- Python
- OpenCV
- YOLOv8
- Ultralytics

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CodeAlpha_ObjectDetection.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Using Webcam

```bash
python object_detection.py
```

For a video file, replace the webcam source with your video file path in the code before running the script.

---

## ⚙️ Workflow

1. Capture video frames
2. Detect objects using YOLOv8
3. Track detected objects
4. Draw bounding boxes
5. Display object labels and tracking IDs
6. Save the processed video
